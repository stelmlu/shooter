Shooter
=======

Shooter is a shoot'em up game using SDL2 and EnTT. It still in progress.

Install and Compile
-------------------

Clone the project

```bash
git clone https://github.com/stelmlu/shooter --recurse-submodules
```

Create a build folder

```bash
cmake -S ./ -B build 
```

Build the game

```bash
cmake --build build --config Release
```

Run

```bash
./Shooter
```
